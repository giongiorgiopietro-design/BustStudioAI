# BustStudioAI

BustStudioAI è una pipeline AI per la preparazione automatica di immagini destinate alla generazione di modelli 3D fotorealistici.

## Obiettivi

- Restauro immagini
- Rimozione sfondo
- Depth Map
- HyperDepth
- Normal Map
- Ottimizzazione per Hyper3D

## Stato del progetto

Versione: v1.0 (In sviluppo)
